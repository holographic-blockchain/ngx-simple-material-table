Basic Table Displaying Object Properties

This shows the default behavior. The columns will bind to a property of the same name.

For values that should be sent to an Angular pipe, use the dataType property. The following are currently supported:

  • number - will use the DecimalPipe
  • date - will use the DatePipe
  • percent - will use the PercentPipe
<hb-mat-table [tableData]="matGridData">
    <hb-mat-table-column name="firstName"></hb-mat-table-column>
    <hb-mat-table-column name="lastName"></hb-mat-table-column>
    <hb-mat-table-column name="age" dataType="number"></hb-mat-table-column>
    <hb-mat-table-column name="birthDate" dataType="date"></hb-mat-table-column>
    <hb-mat-table-column name="skillAssessment" dataType="percent"></hb-mat-table-column>
</hb-mat-table>

tableData: UserInfo[] = ...;
matGridData: MatTableDataSource<UserInfo>;

constructor() {{ '{' }}
    this.matGridData = new MatTableDataSource<UserInfo>(this.tableData);
}

    userId: number | null = null;

Custom Column Rendering

If the data to be rendered in a column needs custom formatting, use a column template. The data variable exposed in this example contains the instance of the data for that row. You can also access other methods or properties defined in your component.

Note that when using a column template, if the rendered value is different than the data source's property value, you will need to define custom sorting or it will not work properly on that column. The sorting will continue to sort based on the original property value.

{{ data.firstName | uppercase }}
 
<hb-mat-table [tableData]="matGridData">
    <hb-mat-table-column name="firstName">
        <div *hbMatTableColumnTemplate="let data">{{ '{{' }} data.firstName | uppercase }}</div>
    </hb-mat-table-column>
    <hb-mat-table-column name="lastName"></hb-mat-table-column>
    <hb-mat-table-column name="age" dataType="number"></hb-mat-table-column>
    <hb-mat-table-column name="birthDate" dataType="date"></hb-mat-table-column>
</hb-mat-table>

If the data to be rendered in a column is not directly available in a single property, or it needs custom formatting, use a column template. Custom Number/Date Formatting

When setting a dataType, a dataFormat property can also be set. This will be passed to the Angular pipe for formatting. The locale used by the pipe will always be the value set globally in your application. If you are unable to set the LOCALE_ID globally, you would need to use a column template.

  • number - will be the digitsInfo parameter of the DecimalPipe
  • date - will be the format parameter of the DatePipe
  • percent - will be the digitsInfo parameter of the PercentPipe
<hb-mat-table [tableData]="matGridData">
    <hb-mat-table-column name="firstName"></hb-mat-table-column>
    <hb-mat-table-column name="lastName"></hb-mat-table-column>
    <hb-mat-table-column name="age" dataType="number" dataFormat="1.1-1"></hb-mat-table-column>
    <hb-mat-table-column name="birthDate" dataType="date" dataFormat="shortDate"></hb-mat-table-column>
    <hb-mat-table-column name="skillAssessment" dataType="percent" dataFormat="1.1-1"></hb-mat-table-column>
</hb-mat-table>

tableData: UserInfo[] = ...;
matGridData: MatTableDataSource<UserInfo>;

    this.matGridData = new MatTableDataSource<UserInfo>(this.tableData);

Only Display Some Columns, Column Display Order

By default, all columns defined will be displayed in the order they are defined. If you want to hide certain columns, or change the order the columns are displayed in, you will need to set the displayedColumns property. This is a string array of column names.

In the example, both the userId and age columns are not displayed. And the order of the firstName and lastName columns are switched. 

<hb-mat-table [tableData]="matGridData"
              [displayedColumns]="customColumnDisplay">
    <hb-mat-table-column name="userId"></hb-mat-table-column>
    <hb-mat-table-column name="firstName"></hb-mat-table-column>
    <hb-mat-table-column name="lastName"></hb-mat-table-column>
    <hb-mat-table-column name="age" dataType="number"></hb-mat-table-column>
    <hb-mat-table-column name="birthDate" dataType="date"></hb-mat-table-column>
</hb-mat-table>

tableData: UserInfo[] = ...;
matGridData: MatTableDataSource<UserInfo>;
customColumnDisplay: string[] = ['lastName', 'firstName', 'birthDate'];

constructor() {{ '{' }}
    this.matGridData = new MatTableDataSource<UserInfo>(this.tableData);
}

Custom Header Text

The column's header text will default to the column name, splitting it by capital letters. A column named 'firstName' displays as 'First Name' in the header row. If the header text should be something else, set the caption property on the column. This does not affect the default sorting behavior. 

<hb-mat-table [tableData]="matGridData">
    <hb-mat-table-column name="firstName"></hb-mat-table-column>
    <hb-mat-table-column name="lastName" caption="Family Name"></hb-mat-table-column>
    <hb-mat-table-column name="age" dataType="number"></hb-mat-table-column>
    <hb-mat-table-column name="birthDate" dataType="date"></hb-mat-table-column>

Footer Row

To add a footer row to the table, set the showFooter property and define a footer template for each relevant column. Only a single footer row is supported.

Average ::
{{ sampleFooterValue | number }}
 
<hb-mat-table [tableData]="matGridData"
              [showFooter]="true">
    <hb-mat-table-column name="firstName">
        <div *hbMatTableFooterTemplate>Average ::</div>
    </hb-mat-table-column>
    <hb-mat-table-column name="lastName"></hb-mat-table-column>
    <hb-mat-table-column name="age" dataType="number">
        <div *hbMatTableFooterTemplate>{{ '{{' }} sampleFooterValue | number }}</div>
    </hb-mat-table-column>
    <hb-mat-table-column name="birthDate" dataType="date"></hb-mat-table-column>
</hb-mat-table>

tableData: UserInfo[] = ...;
matGridData: MatTableDataSource<UserInfo>;
sampleFooterValue: number = 27.3;

Sticky Header Row

To make the header row sticky so it is always visible, set the isHeaderSticky property. 

<hb-mat-table [tableData]="matGridData"
              [isHeaderSticky]="true">
    <hb-mat-table-column name="firstName"></hb-mat-table-column>
    <hb-mat-table-column name="lastName"></hb-mat-table-column>

Fixed Column(s)

To make columns sticky, apply the isStickyStart or isStickyEnd properties to the column.

    <hb-mat-table-column name="userId" [isStickyStart]="true"></hb-mat-table-column>
    <hb-mat-table-column name="firstName"></hb-mat-table-column>
    <hb-mat-table-column name="lastName"></hb-mat-table-column>
