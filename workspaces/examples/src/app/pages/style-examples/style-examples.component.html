Built-in Material Style Classes

Because this wraps the MatTable, all of the style classes added by the Material table will be available. For all classes, please see the Material documentation. You can also inspect the table to view the classes added to various elements. The table below is a partial list of the classes.

When applying custom styles defined in the component, make sure to set the component's encapsulation to None. You should also use a more targeted selector, such as using a wrapper class or setting a class on the hb-mat-table element, so the styles don't inadvertently apply to other components. If you define your styles globally, you will not need to make changes to the component's encapsulation. 

<hb-mat-table [tableData]="matGridData">

Adding a Class to the Component

Setting the attribute on the component is a good way to localize styles that are specific to that one instance. The component's class can then be used with the built-in Material css classes (see above) or with the html elements to define your custom styles for the table.

When applying custom styles defined in the component, make sure to set the component's encapsulation to None. If you define your styles globally, you will not need to make changes to the component's encapsulation.

Setting the TableClass Property

If you have other tables, such as directly using the Material table component or creating your own tables, you may want to add a css class directly to the table and define global styles across your application. To do this, set the tableClass property on the component. The css class(es) will be added directly to the table html element.

If the styles associated with the css class are defined in the component, make sure to set the component's encapsulation to None. If you define your styles globally, you will not need to make changes to the component's encapsulation. 

<hb-mat-table [tableData]="matGridData" tableClass="custom-class">
    <hb-mat-table-column name="firstName"></hb-mat-table-column>
    <hb-mat-table-column name="lastName"></hb-mat-table-column>
    <hb-mat-table-column name="age" dataType="number"></hb-mat-table-column>

Setting the ColumnClass Property

For styling individual columns, set the columnClass property. While the Material table component will automatically add a class based on the column's name, you may want more common styles applied to specific columns. In the example below, a class is added to the age column to right-align it. This class could be defined globally and applied to all number columns.

If the styles associated with the css class are defined in the component, make sure to set the component's encapsulation to None. If you define your styles globally, you will not need to make changes to the component's encapsulation. 

<hb-mat-table [tableData]="matGridData">
    <hb-mat-table-column name="firstName"></hb-mat-table-column>
    <hb-mat-table-column name="lastName"></hb-mat-table-column>
    <hb-mat-table-column name="age" dataType="number" columnClass="column-number"></hb-mat-table-column>
    <hb-mat-table-column name="birthDate" dataType="date"></hb-mat-table-column>
